# Resources for Learning About Hash Tables

This document contains a curated list of resources for learning about hash table implementations from various sources.

## By Company

### Meta (Facebook)

*   [F14: A highly optimized persistent hash map](https://engineering.fb.com/2019/04/25/developer-tools/f14/) - A blog post from Meta Engineering about their highly optimized F14 hash map.

### Google

*   [Abseil: Swiss Tables](https://abseil.io/about/design/swisstables) - An overview of Google's high-performance "Swiss Table" hash map implementation in their Abseil library.

### Microsoft

*   [L4](https://github.com/microsoft/L4) - The GitHub repository for Microsoft's L4, a high-performance hash table implementation.

## By Language/Project

### Python

*   [More compact dictionaries with faster iteration](https://mail.python.org/pipermail/python-dev/2012-December/123028.html) - A Python-dev mailing list discussion about making Python's dictionaries more compact and faster.

### Zend (PHP)

*   [PHP's new hashtable implementation](https://www.npopov.com/2014/12/22/PHPs-new-hashtable-implementation.html) - A blog post explaining the new hash table implementation in PHP.

## Books

### Crafting Interpreters

*   [Hash Tables Chapter](https://craftinginterpreters.com/hash-tables.html) - The chapter on hash tables from the online book "Crafting Interpreters" by Robert Nystrom.
